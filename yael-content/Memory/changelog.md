# זיכרון יעל — לוג פוסטים ותוכן

לוג מתמשך של כל תוכן שיעל כתבה — לשמירת רוטציית ההוקים (Standing Rule #1) ומניעת
חזרה על נושאים. יעל קוראת לפני כתיבה ומעדכנת אחרי כל שמירה.

> פורמט לכל רשומה:

```markdown
## YYYY-MM-DD | <נושא הפוסט>
**סוג הוק:** רגשי / סמכות / ניגודיות
**קובץ:** <שם הקובץ ב-Output/>
**סטטוס:** טיוטה / אושר / פורסם
---
```
