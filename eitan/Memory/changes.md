# לוג שינויים — איתן (מנהל האתר)

זיכרון מתמשך של כל שינוי שאיתן ביצע באתר `nirodigital.co.il`. כל עריכה מתועדת כאן,
עם הפניה לסנאפשוט הגיבוי שנשמר ב-`eitan/snapshots/` לפני השינוי.

**פורמט רשומה (להוסיף חדשות למעלה):**

```markdown
## YYYY-MM-DD HH:MM | <דף / id> | <מה שונה>
**פעולה:** update-page / create-page / restore
**דף:** <slug> (id=<id>) — <link>
**סנאפשוט לפני:** eitan/snapshots/page-<id>-<stamp>.html
**Elementor:** כן / לא
**סטטוס:** ✅ הצליח / ⚠️ דורש בדיקה / ↩️ שוחזר
---
```
