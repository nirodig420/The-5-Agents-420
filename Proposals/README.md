# Proposals — הצעות מוכנות-לאישור של איתן

כאן איתן מניח **הצעה ממוקדת אחת** בכל פעם, מוכנה לאישור ראובן/ניר.
פורמט קובץ: `<YYYY-MM-DD>-<slug>.md`.

כל הצעה כוללת:
- **מה** — השינוי/הנכס המוצע (טיוטה/קוד מדויק).
- **למה** — הרציונל (מילות מפתח, פער מול מתחרה, GEO/AEO).
- **השפעה צפויה** — דירוג / תנועה / המרה.

זרימה: איתן מניח כאן → ראובן מציג לניר → רק אחרי **"אשר"** מתקדמים לביצוע.
