# .claude/commands/ — פקודות מותאמות

**קובץ/תיקייה:** `.claude/commands/` (מכילה כרגע רק `.gitkeep`)
**משויך ל:** ראובן (תשתית התזמור)
**סוג:** תיקיית פקודות מותאמות (placeholder)

## מה זה עושה

תיקייה שנועדה להחזיק **פקודות מותאמות** (custom slash commands) שהצוות יוכל להפעיל. כרגע ריקה — מכילה רק `.gitkeep` כדי לשמר את התיקייה ב-git. כשתתווסף פקודה, היא תופיע כאן כקובץ `.md`.
