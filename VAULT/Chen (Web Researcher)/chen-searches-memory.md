# chen-searches-memory — לוג החיפושים של חן

**קובץ:** `chen/Memory/searches.md`
**משויך ל:** חן (חוקרת הרשת)
**סוג:** זיכרון מתמשך (append-only log)

## מה הקובץ עושה

ה**זיכרון המתמשך** של חן נגד כפילויות. לפני כל חיפוש חן עושה Grep על הקובץ לפי מילות מפתח, כדי לבדוק אם חיפשה נושא דומה ב-30 הימים האחרונים. אחרי כל חיפוש היא מוסיפה entry בפורמט קבוע (נושא, מילות מפתח, שאילתות, מקורות עם דירוג כוכבים, המקור שנבחר, ושם הקובץ ב-Content). כרגע מכיל רק את תבנית הפורמט — אין עדיין חיפושים מתועדים.
