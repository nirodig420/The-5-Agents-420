# yuval-reference — תמונות השראה ליובל

**תיקייה:** `yuval/reference/`
**משויך ל:** יובל (מעצב התמונות)
**סוג:** תיקיית השראה ויזואלית (קלט)

## מה זה עושה

מכילה **תמונות השראה** שמהן יובל מחלץ את שפת הסגנון של הפרויקט לפני יצירת תמונה חדשה. אם התיקייה לא ריקה — יובל קורא אותן ומזהה פלטה, קומפוזיציה, אווירה וסוג רינדור.

**קבצים בתיקייה:**
- `WhatsApp Image 2026-05-24 at 14.40.30.jpeg` — תמונת השראה.
- `תמונה שלי חולצה שחורה.jpeg` — תמונת השראה (דמות בחולצה שחורה).
- `.gitkeep` — שומר התיקייה ב-git.

## קבצים קשורים

- [[agent-yuval]] — סורק את התיקייה הזו בכל בקשה.
- [[yuval-outputs]] — הפלט שנוצר בהשראת קבצים אלה.
