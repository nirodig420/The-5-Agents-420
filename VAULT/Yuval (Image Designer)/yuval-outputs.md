# yuval-outputs — תמונות מוגמרות של יובל

**תיקייה:** `yuval-viz/outputs/`
**משויך ל:** יובל (מעצב התמונות)
**סוג:** תיקיית פלט (תמונות + prompts)

## מה זה עושה

היעד שבו יובל שומר כל תמונה שיצר, בזוג קבצים: `.png` (התמונה) ו-`.txt` באותו שם (ה-prompt המלא ששימש — לצורך איטרציה עתידית). מוסכמת שם: `<YYYY-MM-DD>-<slug>`.

**קבצים בתיקייה:**
- `2026-06-02-cute-dog.png` + `.txt` — גור חמוד, תאורה רכה, אווירה חמה (prompt: "A cute fluffy puppy...").
- `2026-06-02-lunapark-ferriswheel.png` + `.txt` — גלגל ענק בלונה פארק.
- `.gitkeep` — שומר התיקייה ב-git.
