## 2026-06-29 | קונבנציית תיוג UTM (Meta → אתר) — הצעה
**פעולה:** הוכנה הצעת כלל-בית לתיוג UTM של כל קישור מ-FB+IG לאתר (אורגני + ממומן). **לא הועלה דבר חי** — ממתין ל"אשר" של ניר.
**ערוץ:** Facebook + Instagram (שניהם)
**אישור:** ⏳ ממתין — אורגני אחרי "אשר"; ממומן = אישור ידני נפרד (נכנס לתוקף רק עם הפעלת קמפיין).
**כלל הבית (קבוע):**
- ערכים: lowercase, בלי רווחים (מקף `-`), בלי עברית, בלי תווים מיוחדים, עקבי תמיד.
- `utm_source` = `facebook` / `instagram` · `utm_medium` = `social` (אורגני) / `paid-social` (ממומן).
- `utm_campaign` = `bio-link` / `post` / `story` / שם קמפיין · `utm_content` = נושא/תאריך/מודעה (אופציונלי).
- ממומן (Ads Manager → רמת מודעה → "פרמטרים של כתובת URL"):
  `utm_source={{site_source_name}}&utm_medium=paid-social&utm_campaign={{campaign.name}}&utm_content={{ad.name}}`
  ⚠️ קבע שם קמפיין/מודעה (lowercase) לפני השקה — הטוקנים נועלים את השם בזמן הפרסום.
**השפעה צפויה:** מדידת מקור-לידים מדויקת ב-GA4 (source/medium פר-ערוץ) — בלי סיכון תקציבי באורגני.
**נכס:** `Proposals/2026-06-29-utm-tagging-convention.md`
