# Output/ — תוצרי תוכן סופיים

## מוסכמת השמות (מחייבת)
כל קובץ חדש: **`YYYY-MM-DD-<slug>.md`** (כמו ב-`Proposals/` — בלי קבצים לא-מתוארכים).

## מבנה
- **שורש** — פוסטים ותוצרי טקסט (`.md` / `.html` של תוכן).
- **`images/`** — תמונות מוכנות לפרסום (באנרים, פוסטים).
- **`deploy-YYYY-MM-DD/`** — חבילות העלאה לאתר (PHP וכו').
- ויזואל גולמי + prompts → **לא כאן**, אלא ב-`yuval-viz/outputs/`.
- חומר שנושא שלו מת → `_archive/` (לא מוחקים, מעבירים).
