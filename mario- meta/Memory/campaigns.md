# Campaigns — לוג הקמפיינים הממומנים של מריו

לוג מתמשך של כל קמפיין ממומן: מטרה, תקציב, תוצאות, חוות דעת ופעולות אופטימיזציה.
רשומות חדשות למעלה. **תזכורת:** כל שינוי שכרוך בכסף = אישור ידני של ניר לפני ביצוע.

**פורמט רשומה:**
```markdown
## YYYY-MM-DD | <שם הקמפיין>
**מטרה:** <Awareness / Traffic / Leads / Sales / Engagement>
**ערוץ:** Facebook / Instagram / שניהם
**תקציב:** <יומי/כולל> | **תקופה:** <תאריכים>
**תוצאות:** CTR: __ | CPM: __ | CPC: __ | CPL: __ | ROAS: __ | Frequency: __
**חוות דעת:** <Hook / קהל / קריאייטיב / הצעה / משפך — תמצית>
**פעולת אופטימיזציה:** <מה הומלץ>
**סטטוס אישור:** ⏳ ממתין / ✅ אושר ובוצע / ❌ נדחה
---
```
