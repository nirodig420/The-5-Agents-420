# זיכרון יובל — לוג הפקות ויזואל

לוג מתמשך של כל נכס שיובל הפיק — למחזור נכסים קיימים (וידאו = כסף אמיתי) ולמעקב אישורים.
יובל קורא לפני הפקה ומעדכן אחרי כל נכס.

> פורמט לכל רשומה:

```markdown
## YYYY-MM-DD | <נושא הנכס>
**סוג:** תמונה / וידאו (+ עלות משוערת אם וידאו)
**קובץ:** <path ב-outputs/> + <path ה-prompt .txt>
**אושר ע"י ניר:** כן / ממתין
---
```
