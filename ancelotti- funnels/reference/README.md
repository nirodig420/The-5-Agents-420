# Reference — נכסי הזהב של אנצ'לוטי

תיקייה זו מכילה את **נכסי המופת** (swipe file) שאנצ'לוטי וכל הצוות לומדים מהם איך
בונים משפכים ודפי נחיתה ברמה גבוהה. כל נכס כאן הוא הסטנדרט — לא נקודת התחלה לעריכה,
אלא דוגמה ללמוד ממנה ולשאוף אליה (ואז לעקוף).

---

## `niro-funnel-map-LIVE.html` — "מפת המכונה" 🏆

מגנט הלידים האינטראקטיבי שניר וקלוד בנו יחד. **זה הסטנדרט.** כשניר אומר "משהו כמו
דף האפיון" / "כמו מפת המכונה" — לזה הוא מתכוון.

### מה זה
מגנט לידים בדמות **דף אבחון אינטראקטיבי**: הוא מלמד את בעל העסק לזהות איפה המשפך
שלו "מדליף" לקוחות, נותן לו ערך אמיתי (אבחון + צ'קליסט), ובסוף מזמין אותו לפעולה
אחת ברורה — לפנות לניר. זו הדדיות (Cialdini) בפעולה: נותנים ערך לפני שמבקשים.

### המבנה (לומדים ממנו)
1. **באנר לוגו** — מיתוג NIRO בראש.
2. **הוק** — פתיח שעוצר את הגלילה ומגדיר את הכאב (בכיוון חיובי).
3. **מפת 6 התחנות** — אנטומיית המשפך של NIRO, כולל הדגשת **נקודת הדליפה** ⚠️
   (התחנה שבה רוב העסקים מאבדים לקוחות).
4. **צ'קליסט אבחון עצמי** — בעל העסק מסמן לעצמו איפה הוא חלש.
5. **CTA כהה עם QR** — פעולה אחת: "התשובות מגיעות ישירות אליי, ניר."

### האבחון האינטראקטיבי (הלב של הנכס)
מודאל עם **4 שאלות** שמחשב אחוז **"אטימות"** של המשפך (כמה הוא אטום לדליפות),
ומפנה לטופס עם הציון. כל שאלה מצביעה על דליפה אפשרית ומציעה פתרון — **תמיד בכיוון
החיובי של המחשבה** (לא "אתה מפסיד לקוחות" אלא "ככה תתפוס כל פנייה").

### מאפיינים טכניים (ככה בונים נכס דומה)
- **HTML עצמאי לחלוטין** — כל הנכסים מוטמעים (base64): פונטים, QR, לוגו, אייקונים.
  נפתח בכל דפדפן, גם בלי אינטרנט. גודל ~1.4MB, ~175 שורות (השורות ארוכות כי הנכסים מוטמעים).
- **RTL מלא** — עברית, מיושר לימין.
- **פונטים:** Assistant + Secular One.
- **פלטת צבעים:**
  - כחול לוגו `#081255`
  - אדום `#E54242`
  - כתום סיגנל `#FF5A1F`
  - טורקיז `#00C2A8`
  - רקע בהיר `#faf9f5`
- **CTA אחד בלבד** — בלי בלבול, בלי "תחפש באתר".

### איך להתעסק עם הקובץ
- **לעיון/לימוד:** פִּתחו בדפדפן (דאבל-קליק) — עצמאי, לא צריך שרת.
- **לעריכת קוד:** הסגנון ב-`<style>` בראש, הסקריפט (האבחון) ב-`<script>`. הנכסים
  המוטמעים (ה-base64 הארוך) הם פונטים/תמונות — **אל תיגעו בהם**, ערכו רק טקסט/מבנה.
- **זה נכס reference — לא עורכים אותו במקום.** בניית משפך חדש = קובץ חדש ב-`Output/`,
  שמשתמש בנכס הזה כתבנית ומשפר עליו.

> **כלל אצבע לאנצ'לוטי:** קרא את הנכס הזה לפני שאתה בונה מגנט לידים / דף נחיתה חדש.
> שאף לאותה רמה — עצמאי, RTL, אינטראקטיבי כשמתאים, בקול NIRO, עם CTA אחד מדיד.
