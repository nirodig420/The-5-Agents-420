# SEO — דוחות אבחון ומחקרי מילות מפתח

כאן איתן שומר **דוחות אבחון** (audits) ו**מחקרי מילות מפתח** מלאים — החומר הגולמי
שעליו מתבססות ההצעות ב-`Proposals/`.

דוגמאות לתוצרים: ניתוח דירוגים נוכחי, מחקר נפח/תחרות לביטוי, ניתוח פער מול מתחרה,
אודיט טכני (Schema/hreflang/מהירות), תכנית אשכול תוכן (content cluster).

פורמט: `<YYYY-MM-DD>-<topic>.md`. נתונים אמיתיים בלבד — בלי המצאת דירוגים/נפחים.
