# לוג חיפושים — חן

זהו הזיכרון המתמשך של חן. **לפני כל חיפוש** היא עושה Grep על הקובץ הזה לפי מילות
מפתח, כדי לבדוק אם חיפשה נושא דומה ב-30 הימים האחרונים. **אחרי כל חיפוש** היא
מוסיפה entry בפורמט הבא:

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
