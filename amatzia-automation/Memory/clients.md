# זיכרון אמציה — לקוחות ומבני Airtable

כשמוקם לקוח חדש, אמציה בונה לו **Base משלו** ב-Airtable לפי הצורך הספציפי. כאן מתועד
מבנה ה-Base וההקמה לכל לקוח, כדי לתחזק ולשחזר.

> פורמט לכל לקוח:

```markdown
## <שם הלקוח> | הוקם YYYY-MM-DD
**Base:** <שם / Base ID>
**טבלאות:** <Table1, Table2, ...>
**קשרים (Linked Records):** <...>
**Views / Automations פנימיות:** <...>
**אוטומציות מחוברות (Make):** <אילו Scenarios מסנכרנים מול ה-Base>
**הערות:** <צורך ספציפי / התאמות>
---
```
