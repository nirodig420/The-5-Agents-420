# reference/ — בלופרינטים ותבניות אוטומציה

תיקיית השראה/חיקוי של אמציה: תרחישי Make, מבני בוט ב-ManyChat, ותבניות Airtable שהוכיחו
את עצמם — כדי לבנות מהר ועקבי במשימה הבאה.

מה שמים כאן:
- **תרשימי זרימה** של אוטומציות מוצלחות (Trigger → מודולים → פלט).
- **Blueprints של Make** (ייצוא JSON של Scenario) כשרלוונטי.
- **תבניות בוט ManyChat** (שלבי שיחה 1→2→3→4 לסינון לידים).
- **סכמות Airtable** לדוגמה (Base/טבלאות/קשרים) ללקוחות נפוצים.

ריק בהתחלה — מתמלא ככל שאמציה בונה נכסים מוכחים.
